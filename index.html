myFlix-Angular

myFlix Angular is the client side of the myFlix API built with Angular. It allows users to create a profile, view movie details, and add movies to their favorites. myFlix Angular also contains a user profile page that allows users to edit their account details and view their favorite movies list.

Function

  • A user can register or log in on the welcome page.
  • Once logged in users can view all of the movies store in the database.
  • Users can favorite or unfavorite a movie from the movie list view.
  • Users can view movie details by clicking the "Open" button on a movie card.
  • Users can view their profile information, favorite movies list, and update their user data on the profile page.

myFlix-Api

The server-side component of a movies database web application made using MongoDB, Express, and JavaScript/Node.js. This application provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

Function

  • Users can request a list of all movies stored.
  • Users can request information of a movie by title.
  • Users can request data about a genre by genre name.
  • Users can request data about a director by director name.
  • Users can create an account.
  • Users can update user data based on username.
  • Users can add or remove a movie from their favorites list
  • Users can de-register their account.

Meet

Meet is a serverless, progressive web application built with React using a test-driven development technique. This application uses the google calendar API to fetch upcoming events.

Function

  • Users can log in through Google's OAuthorization
  • Users are able to download this app to their desktop or mobile phone.
  • Users can view upcoming events that are stored in a google calendar.
  • Users can change the number of events displayed at a time.
  • Users are able to select a city to search for events in that city.
  • Users can view events offline if they have been viewed previously while online.
